How to check if things are working

Quick health check — paste in any browser:

https://rosesroses.com/api/healthz

Should show: {"status":"ok"}

Live admin dashboard (login required):

https://rosesroses.com/admin

Top tiles show pipeline status, alerts, and overnight stats. Auto-refreshes every 60 seconds.

Check in AWS Console: 1. Sign in to aws.amazon.com (your IAM admin user — NOT root for daily ops) 2. Make sure region "N. Virginia (us-east-1)" is selected top-right 3. Search "ECS" → click rosebase cluster → rosebase-api service 4. Running count should be 1. If it's 0, something crashed.

Common tasks

Deploy new code

Just push to the main branch on GitHub. It deploys automatically in ~10-15 minutes.

Don't push between 2 AM and 6:30 AM ET — nightly jobs are running.

Restart the app (force redeploy of the same code)

aws ecs update-service --cluster rosebase --service rosebase-api --force-new-deployment --region us-east-1

Takes ~5 minutes. No downtime (new container starts before old one stops).

Change a secret (API key, password, etc.)

  1. Go to AWS Console → Secrets Manager
  2. Find the secret (e.g., rosebase/openai-api-key)
  3. Click "Retrieve secret value" → "Edit"
  4. Save the new value
  5. Restart the app (see above) — secrets are loaded at task startup

Check logs

  1. AWS Console → CloudWatch → Log groups → /ecs/rosebase
  2. Click the most recent log stream
  3. Search for error or [job/ to find issues

Triage failed scrapers

  1. Admin dashboard → "Pipeline" sidebar → scraper-health view
  2. Or Admin → Commercial Network → Scraper Health
  3. The Sunday 8 AM auto-retire job clears nurseries broken 60+ days; you handle the fixable ones

Scale the database

If RDS is slow or running out of memory: 1. AWS Console → RDS → rosebase 2. Click "Modify" 3. Change instance class (e.g., db.t4g.medium → db.t4g.large) 4. Choose "Apply immediately" or schedule for maintenance window

What costs money

Current: ~$174/month

The biggest costs are: - Database (RDS): ~$50/mo - Networking (NAT Gateway): ~$40/mo - Compute (ECS Fargate): ~$39/mo - Load balancer (ALB): ~$22/mo - Backups + Performance Insights: ~$14/mo - Storage / misc: ~$9/mo

You'll get an email if the bill goes above $200, and a louder one at $250.

To reduce costs: The database can be scaled down to db.t4g.micro (~$18/mo instead of $50/mo), but the nightly enrichment pipeline might struggle with less memory. Don't drop NAT to one-AZ-only; that's a single point of failure.

Things only a developer can do

These need someone with code-level access: - Fix scraper config when a nursery's website redesigns (CSS selectors break) - Add new nurseries to the scraper roster - Modify pipeline behavior (job order, schedules, thresholds) - Investigate non-trivial errors in logs
